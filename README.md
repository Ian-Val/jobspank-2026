# JobSpank | Job Application Tracker

JobSpank is a client-first web-application that makes keeping track of your job applications a delight.

Developed by [Ian Valeta](https://github.com/Ian-Val/)

## Why?

You can track job applications in a variety of ways: spreadsheets, sticky notes, graph paper, clay tablets, etc... There are already software tools that attempt to solve the problem of helping job applicants track their prospects, but none of them are build specifically for entry-level job seekers.

JobSpank is an open-source job application tracking tool that is designed to provide a small but robust set of tools around tracking job applications to make the experience more delightful for entry-level talent.

## Quick Start

### Clone Repository

```bash
git clone https://github.com/Ian-Val/jobspank-2026.git
```

### Install NPM Packages

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

## Usage

Feel free to start tracking job applications using the live website.



```bash

```

### Submit a pull request

1. **Fork** the repository to your GitHub account.
2. **Push your branch** to your fork:

```bash
git push origin feature/your-branch-name
```

3. Go to the original repository on GitHub and click **Compare & pull request**.
4. Provide a **clear title** and **description** for your pull request.
5. Respond to any review comments. Once approved, your changes will be merged.
